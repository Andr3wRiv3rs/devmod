# Clear Warns

## Usage
`.clearwarns <user>`

Removes all of a user's warnings.

You can add warnings to a user with the [warn](./warn.md) command.

You can list a user's warnings with the [warns](./warns.md) command.

## Permission Requirements
`['KICK_MEMBERS']`
