# Warns

## Usage
`.warns <user>`

Lists a users warnings.

You can add warnings to a user with the [warn](./warn.md) command.

You can clear a user's warnings with the [clear warns](./clearwarns.md) command.

## Permission Requirements
`['KICK_MEMBERS']`
