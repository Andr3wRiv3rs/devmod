# Lock

## Usage
`.lock`

Lock's a channel. Sets permissions for the `@everyone` and `verified` role to `{ 'SEND_MESSAGES' : false }`

You can unlock a channel with the [unlock](./unlock.md) command.

## Permission Requirements
`['MANAGE_CHANNELS']`
